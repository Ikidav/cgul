# تمرين فرز الطلاب

## الوصف:
في هذا التمرين، سنقوم ببناء تطبيق يعرض قائمة من الطلاب ويوفر أزرارًا لفرز الطلاب حسب العمر والاسم.

## المتطلبات:
- يجب عرض قائمة من الطلاب تتضمن اسم الطالب، العمر، وتخصصه.
- يجب توفير أزرار لفرز الطلاب حسب العمر والاسم.
- يجب أن تعمل عملية الفرز بشكل صحيح وفقًا للعمر والاسم.

## التحديات:
- تنفيذ واجهة المستخدم لعرض الطلاب والأزرار بشكل جميل ومنظم.
- التأكد من عملية الفرز تعمل بشكل صحيح وفقًا للمعايير المحددة.

## الخطوات:
1. قم بإنشاء مشروع جديد باستخدام إطار عمل React.
2. قم ببناء مكونات التطبيق المطلوبة مثل قائمة الطلاب والأزرار.
3. اكتب الكود اللازم لعرض الطلاب وتوفير الأزرار للفرز.
4. قم بتجربة التطبيق للتأكد من عملية الفرز بشكل صحيح.
5. أضف أي تحسينات إضافية أو ميزات إذا لزم الأمر.

## كيفية تشغيل التطبيق:
1. قم بتثبيت الأدوات اللازمة باستخدام npm أو yarn: `npm install` أو `yarn install`.
2. ابدأ التطبيق: `npm start` أو `yarn start`.
3. افتح متصفح الويب وانتقل إلى `http://localhost:3000` لرؤية التطبيق.
